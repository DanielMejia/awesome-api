# Awesome API with firebase.

## Version 0.0.0

## Getting started

Before start project is neccesary install the following dependencies in your system:

* [Git](https://git-scm.com/)
* [NodeJs Version 10.14.1](https://nodejs.org)

## Installation
```bash

# Install the NestJs CLI
$ npm i -g @nestjs/cli

# Install the dependencies
$ npm run install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Quality Unit Test, TsLint and Sonarqube

```bash
# unit tests
$ npm run test.

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov

# TsLint analysis
$ npm run lint
```
